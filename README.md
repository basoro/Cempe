# CEMPe panel
Minimal Centos Engine-X (Nginx) MySQL PHP-FPM Panel for mLITE (Medic LITE Indonesia)
- Centos 7
- Nginx 1.24
- MySQL 5.6
- PHP 8.1
- Latest mLITE
- PhpMyAdmin

### How to install?

```bash
curl -o install.sh https://raw.githubusercontent.com/basoro/cempe/master/scripts/install.sh && sh install.sh
````
